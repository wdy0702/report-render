# report-render

## Install
```
npm install report-render --save
```

### Import
```
import ReportRender from 'report-render'

Vue.use(ReportRender)

```

### Use
```
<report-render :reportData="mock"></report-render>

```
### Properties

| Name            | Type              | Default     | Description|
| ---             | ---               | ---         | ---        |
| reportData      | Object            | {}          | 报表数据结构 |
